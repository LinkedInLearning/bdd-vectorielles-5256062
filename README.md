# L'essentiel des bases de données vectorielles

Ce dossier Repository est lié au cours `L'essentiel des bases de données vectorielles`. Le cours entier est disponible sur [LinkedIn Learning][lil-course-url].

![Nom final de la formation][lil-thumbnail-url] 

Animée par Rudi Bruchez, cette formation s’adresse aux développeurs, data scientists et architectes de données souhaitant comprendre et exploiter les bases de données vectorielles, essentielles pour les applications d'intelligence artificielle et de recherche sémantique. Vous apprendrez les principes fondamentaux des vecteurs, comment vectoriser des contenus (texte, image, etc.), et à manipuler des bases de données vectorielles comme Milvus. Grâce à des techniques de recherche par similarité et à la construction de systèmes RAG, vous saurez intégrer efficacement ces technologies dans vos pipelines IA, pour des solutions innovantes et performantes.		

## Instructions

Ce dossier Repository a des branches pour chacune des vidéos du cours. Vous pouvez utiliser le menu des Branches sur GitHub afin d’accéder aux passages qui vous intéressent. Vous pouvez également rajouter `/tree/BRANCH_NAME` à l’URL afin d’accéder à la branche qui vous intéresse. 

## Branches

Les branches sont structurées de manière à correspondre aux vidéos du cours. La convention de nommage est : `CHAPITRE#_VIDEO#`. Par exemple, la branche nommée`02_03` correspond au second chapitre, et à la troisième vidéo de ce chapitre. Certaines branches ont un état de départ et de fin.  
La branche `02_03_d` correspond au code du début de la vidéo.  
La branche `02_03_f` correspond au code à la fin de la vidéo.  
La branche master correspond au code à la fin de la formation.
